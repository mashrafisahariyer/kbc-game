This project is a console-based quiz game inspired by the popular TV show

The game includes:
- 10 multiple-choice questions
- Three difficulty levels: Easy, Medium, Hard
- Increasing prize money for each correct answer
- 50% penalty for wrong answers in Medium and Hard levels
- One lifeline (Second Chance) available in Hard level
- Option to quit the game at any stage

This project demonstrates the use of structures, arrays, loops, conditional
statements, and input validation in C programming.
